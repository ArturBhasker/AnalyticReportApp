Данный проект является консольным приложение с веб-приложением внутри для работы с базой C1NWind (лежит в папке AnalyticReportApp) и формирования html по шаблну
Для запуска прокета нужно запустить AnalyticReportApp.exe с опциями Mark(автомобильный бренд) и Output(результирующий файл).

Шаблон через опции не передаётся так как он жестко привязан к внутренней моделе сервера

например AnalyticReportApp.exe -Mark=Mersedes-Benz -outputFile=AAAAA.html

В проекте всего 1 юнит тест - чисто для демонстрации.

ЗЫ изначально планировал сделать просто веб сервер, но потом решил сделать ближе к заданию, но возможность быстрой переделки на чистый веб сервер оставил.
