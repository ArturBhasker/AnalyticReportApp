Данный проект является консольным веб-приложением для работы с базой C1NWind (лежит в папке AnalyticReportApp) и формирования html ответа
Для запуска прокета нужно запустить AnalyticReportApp.exe  (проект AnalyticReportApp)).

для получения html нужно отправить запрос с фильтром бренда. Например https://localhost:7215/GetCars?brand=bmw

В проекте всего 1 юнит тест - чисто для демонстрации.
